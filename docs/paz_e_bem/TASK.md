# Tarefas Iniciais: Landing Page para Curso Online da Escola PAZ e BEM

## Fase 1: Configuração e Estrutura Base
- [ ] Iniciar o projeto com Next.js (`npx create-next-app@latest`).
- [ ] Configurar o Tailwind CSS como ferramenta de estilização.
- [ ] Criar a estrutura de pastas para componentes (`/components/sections`).
- [ ] Desenvolver o layout global, incluindo Header (com navegação) e Footer.
- [ ] Definir e configurar as fontes e a paleta de cores primárias do projeto.
- [ ] Configurar o Google Tag Manager no projeto.

## Fase 2: Desenvolvimento dos Componentes (Seções)
- [ ] **Componente 1:** Desenvolver a seção Hero, incluindo espaço para vídeo e botão de CTA principal.
- [ ] **Componente 2:** Desenvolver a seção de Prova Social (logos).
- [ ] **Componente 3:** Desenvolver a seção de Conteúdo do Curso (cards ou lista de módulos).
- [ ] **Componente 4:** Desenvolver a seção "Sobre o Instrutor" com foto e biografia do Prof° Antonio Sergio.
- [ ] **Componente 5:** Desenvolver a seção de Depoimentos (preparar para receber vídeos e textos).
- [ ] **Componente 6:** Desenvolver a seção de FAQ com a lógica do "acordeão" funcional.
- [ ] **Componente 7:** Desenvolver a seção de Oferta, com informações de preço, bônus e botão de inscrição.

## Fase 3: Integração e Otimização
- [ ] Implementar a integração com a plataforma de checkout externa (Hotmart/Eduzz).
- [ ] Garantir a responsividade em todos os dispositivos (mobile-first).
- [ ] Implementar `lazy loading` para todas as imagens e iframes de vídeo.
- [ ] Otimizar imagens usando formatos modernos (WebP/AVIF) e compressão adequada.
- [ ] Implementar meta tags para SEO e compartilhamento social.

## Fase 4: Testes e Ajustes
- [ ] Realizar testes de ponta a ponta em diferentes navegadores (Chrome, Firefox, Safari, Edge).
- [ ] Validar a integração com o Google Tag Manager e demais scripts de rastreamento.
- [ ] Testar o fluxo completo de conversão até a página de checkout.
- [ ] Fazer uma auditoria de performance com o Google PageSpeed Insights e Lighthouse.
- [ ] Revisar todo o conteúdo e layout com o cliente/stakeholder.

## Fase 5: Deploy e Entrega Final
- [ ] Configurar o processo de deploy contínuo na Vercel/Netlify.
- [ ] Publicar a versão final e configurar o domínio personalizado.
- [ ] Preparar a documentação técnica para o cliente.
- [ ] Realizar o treinamento de entrega para a equipe da Escola PAZ e BEM.
- [ ] Formalizar a entrega do projeto e assinatura do aceite.
