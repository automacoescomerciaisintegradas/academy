# Prompt Reutilizável: Design Academy

Use este prompt para recriar o estilo visual de referência da página `paz-bem.html` em qualquer projeto.

## Prompt Base

```md
Quero que você atue como diretor(a) de arte e frontend senior e crie uma interface com estética editorial premium inspirada no estilo "Design Academy".

Contexto do projeto:
- Tipo de projeto: [landing page | website institucional | dashboard | página de curso | app web]
- Público-alvo: [descrever]
- Objetivo principal da página: [descrever]
- Conteúdo obrigatório: [seções, textos, CTAs]

Direção visual obrigatória:
1. Paleta quente e sofisticada:
   - --gesso: #EAE6DF
   - --museu: #F4F1EB
   - --carvao: #1C1B1A
   - --taupe: #827C75
   - --terracota: #A84B2B
2. Tipografia:
   - Títulos: Instrument Serif (com itálico em palavras-chave)
   - Corpo e interface: Manrope
3. Linguagem de layout:
   - Grid de 12 colunas
   - Composição assimétrica editorial
   - Blocos com recortes orgânicos e sobreposições sutis
4. Atmosfera:
   - Noise overlay leve
   - Textura de fundo com blend suave
   - Grid de linhas finas de baixa opacidade
   - Elemento tipográfico gigante de fundo com opacidade muito baixa
5. Interações:
   - Header com efeito de blur ao scroll
   - Hovers elegantes (mudança de cor, deslocamento sutil)
   - Botões cápsula com borda terracota
   - Reveal animations com GSAP (entrada de hero, fade-up, clip reveal)
6. Responsividade:
   - Experiência impecável em mobile
   - Desativar cursor customizado e efeitos pesados em dispositivos touch

Requisitos de qualidade:
- Não entregar layout genérico.
- Priorizar legibilidade e hierarquia visual.
- Evitar exagero de animações.
- Resultado final deve parecer artesanal, refinado e autoral.

Entregue:
1. Estrutura HTML semântica.
2. CSS organizado por blocos (tokens, layout, componentes, estados, responsivo).
3. JS para interações e animações.
4. Comentários curtos explicando decisões não óbvias.
```

## Versão Curta (uso rápido)

```md
Crie uma página com estética editorial premium "Design Academy": paleta gesso/museu/carvão/taupe/terracota, títulos Instrument Serif, corpo Manrope, grid assimétrica 12 colunas, textura + noise + linhas finas de fundo, botões cápsula terracota, hover refinado, header com blur no scroll, animações GSAP suaves, mobile impecável com fallback para touch. Evite aparência de template genérico.
```

## Checklist de Revisão

- [ ] Paleta aplicada corretamente
- [ ] Tipografia correta (Instrument Serif + Manrope)
- [ ] Layout com personalidade editorial (não genérico)
- [ ] Interações suaves e úteis
- [ ] Boa leitura em desktop e mobile
- [ ] Performance aceitável (sem excesso de efeitos)
